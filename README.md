# younghye
기획 의도
커뮤니티 + 소비분석 + 지출비교 사이트

주요 기능
- 한달 단위 지출 입력
- 분석 그래프
- 정보 공유


개발환경
- 개발언어 : 자바, 제이쿼리, 자바스크립트
- 개발환경 : 오라클, 이클립스
- 프레임워크 : 부트스트랩
- API : chart.js , TOAST UI
